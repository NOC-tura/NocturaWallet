# Step 18: Transparent Send Screen Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the transparent send screen with recipient address input (paste/QR/address book), token selector, string-based BigInt amount parsing, priority fee toggle, SPL token account check (ATA creation warning), transaction simulation with error blocking, confirmation screen with combined network fee, double-submit guard, and post-send navigation to TransactionStatusScreen.

**Architecture:** SendScreen is a multi-step component: (1) input fields → (2) simulate → (3) confirm → (4) send. A `parseTokenAmount` utility converts decimal strings to BigInt without float. The confirmation step is gated by simulation success — if simulation fails, CONFIRM button is blocked and the error is shown. Post-send navigates to TransactionStatusScreen (built in Step 19). QR scanning uses a validation function that detects Solana addresses, Solana Pay URIs, shielded addresses, and rejects invalid/non-Solana inputs.

**Tech Stack:** React Native, @solana/web3.js, transaction builder, simulation module, PinPad-style amount input, BigInt

Create `src/utils/parseTokenAmount.ts`:

```typescript
/**
 * Parse a decimal string to BigInt smallest unit.
 * NEVER uses parseFloat — string-based parsing only.
 *
 * "1.5" with 9 decimals → 1_500_000_000n
 * "0.000000001" with 9 decimals → 1n
 */
export function parseTokenAmount(input: string, decimals: number): bigint {
  if (!input || input.trim() === '') return 0n;

  const trimmed = input.trim();

  // Reject negative and non-numeric
  if (trimmed.startsWith('-')) throw new Error('Negative amounts not allowed');
  if (!/^\d*\.?\d*$/.test(trimmed)) throw new Error('Invalid amount format');

  const parts = trimmed.split('.');
  const wholePart = parts[0] || '0';
  const fracPart = parts[1] || '';

  // Reject too many decimal places
  if (fracPart.length > decimals) {
    throw new Error(`Too many decimal places (max ${decimals})`);
  }

  // Pad fractional part to full decimals
  const paddedFrac = fracPart.padEnd(decimals, '0');

  // Combine: whole * 10^decimals + fractional
  const wholeUnits = BigInt(wholePart) * BigInt(10 ** decimals);
  const fracUnits = BigInt(paddedFrac);

  return wholeUnits + fracUnits;
}

/**
 * Format BigInt smallest unit to display string.
 * Inverse of parseTokenAmount.
 */
export function formatTokenAmount(amount: bigint, decimals: number): string {
  const divisor = BigInt(10 ** decimals);
  const whole = amount / divisor;
  const remainder = amount % divisor;

  if (remainder === 0n) return whole.toString();

  let remStr = remainder.toString().padStart(decimals, '0');
  remStr = remStr.replace(/0+$/, ''); // trim trailing zeros

  return `${whole}.${remStr}`;
}
```

Create `src/utils/validateAddress.ts`:
```typescript
export type AddressType = 'solana' | 'solana_pay' | 'shielded' | 'non_solana' | 'invalid';

export interface AddressValidation {
  type: AddressType;
  address?: string;
  amount?: string;
  token?: string;
  error?: string;
}

const SOLANA_ADDRESS_REGEX = /^[1-9A-HJ-NP-Za-km-z]{32,44}$/;
const ETH_ADDRESS_REGEX = /^0x[0-9a-fA-F]{40}/;
const BTC_ADDRESS_REGEX = /^(bc1|[13])[a-zA-HJ-NP-Z0-9]{25,62}$/;

export function validateRecipientInput(input: string): AddressValidation {
  const trimmed = input.trim();
  if (!trimmed) return {type: 'invalid', error: 'Address is required'};

  // Shielded address (noc1...)
  if (trimmed.startsWith('noc1')) {
    return {type: 'shielded', address: trimmed};
  }

  // Solana Pay URI
  if (trimmed.startsWith('solana:')) {
    return parseSolanaPay(trimmed);
  }

  // Non-Solana detection
  if (ETH_ADDRESS_REGEX.test(trimmed)) {
    return {type: 'non_solana', error: 'This is not a Solana address'};
  }
  if (BTC_ADDRESS_REGEX.test(trimmed)) {
    return {type: 'non_solana', error: 'This is not a Solana address'};
  }

  // Solana address
  if (SOLANA_ADDRESS_REGEX.test(trimmed)) {
    return {type: 'solana', address: trimmed};
  }

  return {type: 'invalid', error: 'Invalid recipient address'};
}

function parseSolanaPay(uri: string): AddressValidation {
  try {
    const withoutScheme = uri.replace('solana:', '');
    const [address, queryString] = withoutScheme.split('?');

    if (!address || !SOLANA_ADDRESS_REGEX.test(address)) {
      return {type: 'invalid', error: 'Invalid Solana Pay address'};
    }

    const params = new URLSearchParams(queryString || '');
    return {
      type: 'solana_pay',
      address,
      amount: params.get('amount') ?? undefined,
      token: params.get('spl-token') ?? undefined,
    };
  } catch {
    return {type: 'invalid', error: 'Invalid Solana Pay URI'};
  }
}
```

Create `src/utils/formatAddress.ts`:

```typescript
/**
 * Truncate a Solana address for display: first 6 + "..." + last 4.
 */
export function formatAddress(address: string): string {
  if (address.length <= 12) return address;
  return `${address.slice(0, 6)}...${address.slice(-4)}`;
}
```
