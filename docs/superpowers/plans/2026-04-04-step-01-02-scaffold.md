# Step 1–2: Project Scaffold Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Transform the bare RN 0.84.1 scaffold into a fully configured project with polyfill bootstrap, NativeWind v4, navigation tree, Zustand stores, dual MMKV instances, env config, design tokens, and error taxonomy — everything needed before any screens are built.

**Architecture:** Two MMKV instances (public unencrypted + secure encrypted lazy-init). Five Zustand stores split by domain (wallet, session, publicSettings, secureSettings, presale, shielded) with a `useSettings()` facade hook. React Navigation v7 with RootStack → MainTabs (4 tabs) + OnboardingStack + modal screens. NativeWind v4 with Noctura design tokens. Polyfill bootstrap in strict order (get-random-values → url-polyfill → text-encoding).

**Tech Stack:** React Native 0.84.1, TypeScript 5.x strict, NativeWind v4 / Tailwind 3, React Navigation v7, Zustand, react-native-mmkv, react-native-config, @tanstack/react-query v5, react-native-reanimated v3, react-native-gesture-handler, react-native-safe-area-context

**Validated decisions:** See `docs/superpowers/specs/2026-04-04-architecture-validation-design.md`

import {create} from 'zustand';

const DEFAULT_TIMEOUT_MINUTES = 5;

interface SessionState {
  isUnlocked: boolean;
  unlockedAt: number | null;
  lastActiveAt: number | null;
  sessionExpiresAt: number | null;

  unlock: () => void;
  lock: () => void;
  touchActivity: () => void;
  isExpired: () => boolean;
}

export const useSessionStore = create<SessionState>((set, get) => ({
  isUnlocked: false,
  unlockedAt: null,
  lastActiveAt: null,
  sessionExpiresAt: null,

  unlock: () => {
    const now = Date.now();
    set({
      isUnlocked: true,
      unlockedAt: now,
      lastActiveAt: now,
      sessionExpiresAt: now + DEFAULT_TIMEOUT_MINUTES * 60 * 1000,
    });
  },

  lock: () =>
    set({
      isUnlocked: false,
      unlockedAt: null,
      lastActiveAt: null,
      sessionExpiresAt: null,
    }),

  touchActivity: () => set({lastActiveAt: Date.now()}),

  isExpired: () => {
    const {sessionExpiresAt} = get();
    if (!sessionExpiresAt) return true;
    return Date.now() > sessionExpiresAt;
  },
}));
