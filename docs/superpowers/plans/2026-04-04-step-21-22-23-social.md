**Tech Stack:** react-native-mmkv (secure), Zustand stores, Solana queries, Share API

## Task 3: Referral Code Generator Utility

### `src/utils/generateReferralCode.ts`:
```typescript
export function generateReferralCode(walletAddress: string): string {
  // Simple hash: take first 4 chars of hex hash of address
  let hash = 0;
  for (let i = 0; i < walletAddress.length; i++) {
    hash = ((hash << 5) - hash + walletAddress.charCodeAt(i)) | 0;
  }
  const hex = Math.abs(hash).toString(16).toUpperCase().padStart(4, '0').slice(0, 4);
  return `NOC-${hex}`;
}
```

Commit: `git commit -m "feat: referral code generator (NOC-XXXX from wallet address hash)"`
